Project Goals

Our primary aim is to make digital systems more accessible and inclusive by developing AI-Autostart, a touch-free interaction platform that enables automatic system activation and adaptive interfaces for users with different accessibility needs. Specifically, our goals were as follows:

  • Develop a Windows system tray application that runs in the background and manages sensing and activation logic while minimising CPU, memory, and power usage
  • Enable touch-free system activation through reliable user presence detection and gesture recognition, including sustained hand-raising to reduce false positives in busy environments
  • Design an adaptive self-service kiosk interface that automatically adjusts layout, size, contrast, and complexity for wheelchair users, visually impaired users, neurodivergent users, and users with limited motor mobility without requiring manual accessibility selection
  • Implement a fully playable no-GUI game for blind and visually impaired users that launches automatically using computer vision for detection, gesture-based input for navigation, and audio-only output for interaction
  • Integrate a local Large Language Model to dynamically generate and narrate game content, ensuring responsive and personalised interaction
  • Deliver three Windows executables that demonstrate the complete AI-Autostart pipeline and allow the system to run independently on standard machines

With AI-Autostart, we aim to transform traditional interaction with digital systems by removing the need for manual setup and interface navigation, creating a more inclusive, automated, and accessible computing experience for users with diverse needs.
