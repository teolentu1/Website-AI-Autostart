AI-Autostart & No-GUI Game - Hand Gesture Recognition

Google's library MediaPipe is used for hand gesture recognition. This library is used in both AI-Autostart and the No-GUI player. In both cases the corresponding video gesture recognition classes create MediaPipe's GestureRecognizer with the following options, allowing for real-time input using the OpenCV library to capture webcam input. It is important to notice that method self._result_callback is injected as a parameter — this method is run asynchronously by the GestureRecognizer.

def _create_recognizer(self):
   options = GestureRecognizerOptions(
       base_options=BaseOptions(model_asset_path=self.model_path),
       running_mode=RunningMode.LIVE_STREAM,
       result_callback=self._result_callback,
   )
   return GestureRecognizer.create_from_options(options)

The implementation of _result_callback differs between the two systems.

No-GUI Player

In the No-GUI player, the public-facing function is get_gesture. This is a blocking function whose functionality has been inspired by Python's built-in input function, except that instead of returning text typed into the terminal it returns the gesture shown by the user. Its only argument is gestures_to_spot — in most cases the No-GUI player is looking for a subset of all gestures, and this is the list of hand gestures it should be watching for.

def get_gesture(self, gestures_to_spot: list[EnumGesture]) -> EnumGesture:
   """
   Start the gesture recognition process and return the first gesture detected that is in the gestures_to_spot list.
   :param gestures_to_spot:
   :return:
   """
   self._reset(gestures_to_spot)
   self._start_recognition()
   return self._get_last_gesture()

Since _result_callback is run asynchronously, the VideoGestureRecogniser state has the attribute _is_running. As soon as _result_callback recognises a gesture from the _gestures_to_spot list, it stops the main loop by setting _is_running to False.

def _result_callback(self, result: GestureRecognizerResult, output_image: mp.Image, timestamp_ms: int):
   """
   Run for each picture analysed by the recognizer. If a gesture to spot is detected, stop the recognition.
   """
   if len(result.gestures) < 1:
       # then no hand detected
       return
   # set the last gesture
   self._last_gesture_category = result.gestures[0][0].category_name
   self._last_handedness = result.handedness[0][0].category_name

   # victory set as a special case for stopping the loop
   if self._get_last_gesture() in self._gestures_to_spot:
       self._stop()

AI-Autostart

In AI-Autostart, the VideoGestureRecogniser class works as a single-responsibility class, implementing the Observer pattern. When a gesture is recognised, its subscribers — PowerManager and GestureController — are updated.
